CSS Display Property

The CSS display property specifies the display behavior of an element. It determines how an element is displayed on the web page.

Common Display Values

  • block: The element is displayed as a block-level element. It starts on a new line and takes up the full width available. Example: <div>, <p>.
  • inline: The element is displayed as an inline element. It does not start on a new line and only takes up as much width as necessary. Example: <span>, <a>.
  • inline-block: The element is displayed as an inline-level block container. It allows setting width and height, but flows with text like an inline element.
  • none: The element is not displayed at all (it is removed from the document flow).
  • flex: The element becomes a flex container, allowing for flexible layouts using flexbox properties.
  • grid: The element becomes a grid container, enabling grid layout capabilities.

Example Usage


    .block-example {
        display: block;
    }

    .inline-example {
        display: inline;
    }

    .inline-block-example {
        display: inline-block;
    }

    .none-example {
        display: none;
    }

    .flex-example {
        display: flex;
    }

    .grid-example {
        display: grid;
    }

Understanding the display property is essential for controlling the layout and appearance of web pages using CSS.
