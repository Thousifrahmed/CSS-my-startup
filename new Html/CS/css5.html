Fonts, Text & Color Properties

Fonts

CSS provides several properties for controlling the font of text:

  • font-family: Specifies the font for an element.
  • font-size: Specifies the size of the font.
  • font-weight: Specifies the weight (boldness) of the font.
  • font-style: Specifies the style of the font (normal, italic, etc.).

Text Properties

CSS also provides properties for controlling the text:

  • text-align: Specifies the horizontal alignment of text.
  • text-decoration: Specifies the decoration added to text (underline, overline, etc.).
  • text-transform: Specifies the capitalization of text.
  • line-height: Specifies the height of a line box.

Color Properties

CSS allows you to set the color of text and other elements:

  • color: Specifies the color of text.
  • background-color: Specifies the background color of an element.
  • border-color: Specifies the color of an element's border.
