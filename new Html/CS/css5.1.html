Colors

colours can be represented in css in various ways including:

  1. Named Colors: e.g., "red", "blue", "green".
  2. Hexadecimal: e.g., #FF5733, #33FF57.
  3. RGB: e.g., rgb(255, 87, 51).
  4. RGBA: e.g., rgba(255, 87, 51, 0.5) (includes alpha for transparency).
  5. HSL: e.g., hsl(9, 100%, 60%).
  6. HSLA: e.g., hsla(9, 100%, 60%, 0.5) (includes alpha for transparency).
